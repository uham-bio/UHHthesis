# Package index

## R Markdown output formats

Output format functions for use in the YAML header of `.Rmd` files. Each
function returns a bookdown output format object that can be passed to
[`bookdown::render_book()`](https://pkgs.rstudio.com/bookdown/reference/render_book.html).

- [`thesis_pdf_de()`](https://uham-bio.github.io/UHHthesis/reference/thesis_pdf_de.md)
  [`thesis_pdf_en()`](https://uham-bio.github.io/UHHthesis/reference/thesis_pdf_de.md)
  : Convert R Markdown to a PDF/LaTeX or thesis document in German and
  English
- [`thesis_word_de()`](https://uham-bio.github.io/UHHthesis/reference/thesis_word_de.md)
  [`thesis_word_en()`](https://uham-bio.github.io/UHHthesis/reference/thesis_word_de.md)
  : Convert R Markdown to a Word (docx) thesis document in German and
  English
